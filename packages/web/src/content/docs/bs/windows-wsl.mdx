Iako Blackbox AI moze raditi direktno na Windowsu, preporucujemo [Windows Subsystem for Linux (WSL)](https://learn.microsoft.com/en-us/windows/wsl/install) za najbolje iskustvo. WSL daje Linux okruzenje koje glatko radi sa Blackbox AI funkcijama.

:::tip[Zašto WSL?]
WSL nudi bolje performanse fajl sistema, punu terminalsku podrsku i kompatibilnost s razvojnim alatima na koje se Blackbox AI oslanja.
:::

---

## Postavljanje

<Steps>

1. **Instalirajte WSL**

   Ako vec niste, [instalirajte WSL](https://learn.microsoft.com/en-us/windows/wsl/install) prema zvanicnom Microsoft vodicu.

2. **Instalirajte Blackbox AI u WSL-u**

   Kad je WSL spreman, otvorite WSL terminal i instalirajte Blackbox AI jednom od [metoda instalacije](/docs/).

   ```bash
   curl -fsSL https://www.blackbox.ai/install | bash
   ```

3. **Koristite Blackbox AI iz WSL-a**

   Idite u direktorij projekta (Windows fajlovima pristupate preko `/mnt/c/`, `/mnt/d/` itd.) i pokrenite Blackbox AI.

   ```bash
   cd /mnt/c/Users/YourName/project
   blackbox ai
   ```

</Steps>

## Desktop aplikacija + WSL Server

Ako preferirate Blackbox AI Desktop aplikaciju, ali zelite da server radi u WSL-u:

1. **Pokrenite server u WSL-u** sa `--hostname 0.0.0.0` da dozvolite vanjske konekcije:

   ```bash
   blackbox ai serve --hostname 0.0.0.0 --port 4096
   ```

2. **Povezite Desktop aplikaciju** na `http://localhost:4096`

:::note
Ako `localhost` ne radi u vasem setupu, povezte se preko WSL IP adrese (u WSL-u: `hostname -I`) i koristite `http://<wsl-ip>:4096`.
:::

:::caution
Kada koristite `--hostname 0.0.0.0`, postavite `BLACKBOX_AI_SERVER_PASSWORD` da zastitite server.

```bash
BLACKBOX_AI_SERVER_PASSWORD=your-password blackbox ai serve --hostname 0.0.0.0
```

:::

## Web klijent + WSL

Za najbolje web iskustvo na Windowsu:

1. **Pokrenite `blackbox ai web` u WSL terminalu** umjesto u PowerShell-u:

   ```bash
   blackbox ai web --hostname 0.0.0.0
   ```

2. **Otvorite iz Windows browsera** na `http://localhost:<port>` (Blackbox AI ispisuje URL)

Pokretanje `blackbox ai web` iz WSL-a osigurava ispravan pristup fajl sistemu i terminalsku integraciju, a i dalje je dostupno iz Windows browsera.

## Pristup Windows fajlovima

WSL moze pristupiti svim Windows fajlovima kroz `/mnt/` direktorij:

- Disk `C:` → `/mnt/c/`
- Disk `D:` → `/mnt/d/`
- I tako dalje...

Primjer:

```bash
cd /mnt/c/Users/YourName/Documents/project
blackbox ai
```

:::tip
Za najgladje iskustvo, razmislite da klonirate/kopirate repo u WSL fajl sistem (npr. pod `~/code/`) i tu pokrenete Blackbox AI.
:::

---

## Savjeti

- Drzite Blackbox AI u WSL-u za projekte na Windows diskovima - pristup fajlovima je jednostavan
- Koristite VS Code [WSL ekstenziju](https://code.visualstudio.com/docs/remote/wsl) uz Blackbox AI za integrisan tok rada
- Vase Blackbox AI konfiguracije i sesije cuvaju se u WSL okruzenju na `~/.local/share/blackbox ai/`
