Blackbox AI는 VS Code, Cursor, 또는 터미널을 지원하는 모든 IDE와 통합됩니다. 시작하려면 터미널에서 `blackbox ai`를 실행하세요.

---

## 사용법

- **Quick Launch**: `Cmd+Esc` (Mac) 또는 `Ctrl+Esc` (Windows/Linux)를 사용해 분할 터미널 뷰에서 Blackbox AI를 열거나, 이미 실행 중인 터미널 세션으로 포커스하세요.
- **New Session**: `Cmd+Shift+Esc` (Mac) 또는 `Ctrl+Shift+Esc` (Windows/Linux)를 사용해 기존 세션이 열려 있어도 새 Blackbox AI 터미널 세션을 시작하세요. UI의 Blackbox AI 버튼을 클릭해도 됩니다.
- **Context Awareness**: 현재 선택 영역이나 탭을 Blackbox AI와 자동으로 공유합니다.
- **File Reference Shortcuts**: `Cmd+Option+K` (Mac) 또는 `Alt+Ctrl+K` (Linux/Windows)를 사용해 파일 참조를 삽입하세요. 예: `@File#L37-42`.

---

## 설치

VS Code와 Cursor, Windsurf, VSCodium 같은 인기 포크에 Blackbox AI를 설치하려면:

1. VS Code를 여세요.4
2. 통합 터미널을 여세요.
3. `blackbox ai`를 실행하세요. 확장 프로그램이 자동으로 설치됩니다.

반면 TUI에서 `/editor` 또는 `/export`를 실행할 때 자체 IDE를 사용하려면 `export EDITOR="code --wait"`를 설정해야 합니다. [자세히 알아보기](/docs/tui/#editor-setup).

### 수동 설치

Extension Marketplace에서 **Blackbox AI**를 검색한 다음 **Install**을 클릭하세요.

---

### 문제 해결

확장이 자동으로 설치되지 않는 경우:

- 통합 터미널에서 `blackbox ai`를 실행하고 있는지 확인하세요.
- IDE용 CLI가 설치되어 있는지 확인하세요.
  - VS Code: `code` command
  - Cursor: `cursor` command
  - Windsurf: `windsurf` command
  - VSCodium: `codium` command
  - 설치되어 있지 않다면 `Cmd+Shift+P` (Mac) 또는 `Ctrl+Shift+P` (Windows/Linux)를 실행하고 "Shell Command: Install 'code' command in PATH"(또는 IDE에 맞는 동등한 명령)를 검색하세요.
- VS Code에 확장 프로그램 설치 권한이 있는지 확인하세요.
