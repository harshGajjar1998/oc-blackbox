blackbox ai может работать как веб-приложение в вашем браузере, обеспечивая такой же мощный опыт кодирования AI без необходимости использования терминала.

![blackbox ai Web — новый сеанс](../../../assets/web/web-homepage-new-session.png)

## Начало работы

Запустите веб-интерфейс, выполнив:

```bash
blackbox ai web
```

Это запустит локальный сервер `127.0.0.1` со случайным доступным портом и автоматически откроет blackbox ai в браузере по умолчанию.

:::caution
Если `BLACKBOX_AI_SERVER_PASSWORD` не установлен, сервер будет незащищен. Это подходит для локального использования, но его следует настроить для доступа к сети.
:::

:::tip[Пользователи Windows]
Для получения наилучших результатов запустите `blackbox ai web` из [WSL](/docs/windows-wsl), а не из PowerShell. Это обеспечивает правильный доступ к файловой системе и интеграцию терминала.
:::

## Конфигурация

Вы можете настроить веб-сервер с помощью CLI-флагов или в файле [config file](/docs/config).

### Порт

По умолчанию blackbox ai выбирает доступный порт. Вы можете указать порт:

```bash
blackbox ai web --port 4096
```

### Имя хоста

По умолчанию сервер привязывается к `127.0.0.1` (только локальный хост). Чтобы сделать blackbox ai доступным в вашей сети:

```bash
blackbox ai web --hostname 0.0.0.0
```

При использовании `0.0.0.0` blackbox ai будет отображать как локальные, так и сетевые адреса:

```
  Local access:       http://localhost:4096
  Network access:     http://192.168.1.100:4096
```

### Обнаружение mDNS

Включите mDNS, чтобы ваш сервер был доступен для обнаружения в локальной сети:

```bash
blackbox ai web --mdns
```

Это автоматически устанавливает имя хоста `0.0.0.0` и объявляет сервер как `blackbox ai.local`.

Вы можете настроить доменное имя mDNS для запуска нескольких экземпляров в одной сети:

```bash
blackbox ai web --mdns --mdns-domain myproject.local
```

### CORS

Чтобы разрешить дополнительные домены для CORS (полезно для пользовательских интерфейсов):

```bash
blackbox ai web --cors https://example.com
```

### Аутентификация

Чтобы защитить доступ, установите пароль, используя переменную среды `BLACKBOX_AI_SERVER_PASSWORD`:

```bash
BLACKBOX_AI_SERVER_PASSWORD=secret blackbox ai web
```

Имя пользователя по умолчанию — `blackbox ai`, но его можно изменить с помощью `BLACKBOX_AI_SERVER_USERNAME`.

## Использование веб-интерфейса

После запуска веб-интерфейс предоставляет доступ к вашим сеансам blackbox ai.

### Сессии

Просматривайте свои сеансы и управляйте ими с главной страницы. Вы можете видеть активные сеансы и начинать новые.

![blackbox ai Web — активный сеанс](../../../assets/web/web-homepage-active-session.png)

### Статус сервера

Нажмите «Просмотреть серверы», чтобы просмотреть подключенные серверы и их статус.

![blackbox ai Web — см. Серверы](../../../assets/web/web-homepage-see-servers.png)

## Подключение терминала

Вы можете подключить TUI терминала к работающему веб-серверу:

```bash
# Start the web server
blackbox ai web --port 4096

# In another terminal, attach the TUI
blackbox ai attach http://localhost:4096
```

Это позволяет вам одновременно использовать веб-интерфейс и терминал, используя одни и те же сеансы и состояние.

---

## Конфигурационный файл

Вы также можете настроить параметры сервера в файле конфигурации `blackbox ai.json`:

```json
{
  "server": {
    "port": 4096,
    "hostname": "0.0.0.0",
    "mdns": true,
    "cors": ["https://example.com"]
  }
}
```

CLI-флаги имеют приоритет над настройками файла конфигурации.
