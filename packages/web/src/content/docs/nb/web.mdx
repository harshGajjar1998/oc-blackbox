Blackbox AI kan kjøres som en nettapplikasjon i nettleseren din, og gir den samme kraftige AI-kodingsopplevelsen uten at du trenger en terminal.

![Blackbox AI Web - Ny økt](../../../assets/web/web-homepage-new-session.png)

## Komme i gang

Start web-grensesnittet ved å kjøre:

```bash
blackbox ai web
```

Dette starter en lokal server på `127.0.0.1` med en tilfeldig tilgjengelig port og åpner automatisk Blackbox AI i standard nettleser.

---

## Konfigurasjon

Du kan konfigurere webserveren ved å bruke kommandolinjeflagg eller i [konfigurasjonsfilen](/docs/config).

### Port

Som standard velger Blackbox AI en tilgjengelig port. Du kan spesifisere en port:

```bash
blackbox ai web --port 4096
```

### Vertsnavn

Som standard binder serveren seg til `127.0.0.1` (kun localhost). Slik gjør du Blackbox AI tilgjengelig på nettverket ditt:

```bash
blackbox ai web --hostname 0.0.0.0
```

---

### Autentisering

For å beskytte tilgang, angi et passord ved hjelp av miljøvariabelen `BLACKBOX_AI_SERVER_PASSWORD`:

```bash
BLACKBOX_AI_SERVER_PASSWORD=secret blackbox ai web
```

Brukernavnet er satt til `blackbox ai` som standard, men kan endres med `BLACKBOX_AI_SERVER_USERNAME`.

## Bruke webgrensesnittet

Når den er startet, gir nettgrensesnittet tilgang til dine Blackbox AI-økter.

### Sesjoner

Se og administrer øktene dine fra hjemmesiden. Du kan se aktive økter og starte nye.

![Blackbox AI Web - Active Session](../../../assets/web/web-homepage-active-session.png)

### Serverstatus

Klikk på "Se servere" for å se tilkoblede servere og deres status.

![Blackbox AI Web - Se servere](../../../assets/web/web-homepage-see-servers.png)

## Koble til en terminal

Du kan koble en terminal TUI til en kjørende webserver:

```bash
# Start the web server
blackbox ai web --port 4096

# In another terminal, attach the TUI
blackbox ai attach http://localhost:4096
```

Dette lar deg bruke både nettgrensesnittet og terminalen samtidig, og deler samme økter og tilstand.

---

## Konfigurasjonsfil

Du kan også konfigurere serverinnstillinger i `blackbox ai.json` konfigurasjonsfilen:

```json
{
  "server": {
    "port": 4096,
    "hostname": "0.0.0.0",
    "mdns": true,
    "cors": ["https://example.com"]
  }
}
```

Kommandolinjeflagg har forrang over konfigurasjonsfilinnstillinger.
